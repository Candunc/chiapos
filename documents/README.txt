Notes on proof_of_space.md and json_api.md and their build processes:

proof_of_space.html and json_api.html is generated from their respective .md files using "Markdown All in One" plugin for Visual Studio Code. The latest version of the plugin is 3.4.0 as of 2021-05-29.

The following settings should be changed to generate the HTML in a consistent way:
- Set "Absolute Img Path" to False.
- Set "Downcase Link" to False.
